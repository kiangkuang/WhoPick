# WhoPick

## Usage

1. Install dependencies
    ```
    npm install
    ```
2. Set up environment variables file
    ```
    npm run setup
    ```
3. Open `.env` file and replace `<token>` and `<db url>` with the real bot token and MySQL DB URL

4. Create DB
    ```
    npm run create-db
    ```
5. Run DB migrations
    ```
    npm run migrate
    ```
6. Development
    ```
    npm run dev
    ```
7. Production
    ```
    npm run start
    ```
